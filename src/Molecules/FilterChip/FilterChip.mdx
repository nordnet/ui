# FilterChip

This component will render a hidden checkbox in the shape of a "pill" alongside an optional icon.
It can be used to quickly toggle filters or toggle UI element on and off.

<Props of={FilterChip} />

## Usage

Basic usage

```javascript

  label='label'
  value='value'
```
